Bug ID - 01 
Test Case ID - TC-60

Title - Unauthenticated user can add products to cart

Description - The user is able to add products to the shopping cart outside of the website's session.

Preconditions - User is not logged in 

Steps to reproduce: 
    1. Navigate to the Home page
    2. Click on the "Add to cart" button on any product

Expected result - The "Add to cart button should not be available to users which aren't logged in

Actual result - The user is able to add any products to the cart without being logged in

Environment: OS - Windows 10; browser - Firefox; Version - 149

Priority - High

Severity - High
