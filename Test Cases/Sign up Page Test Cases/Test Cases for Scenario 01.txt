Test Case ID - TC-01 

Title - Verify that a user can successfully Sign up with a valid email and password

Preconditions - User not logged in; User does not have an account

Test Data: email - testing_user@gmail.com; password - TestingPassword123 

Test Steps: 
    1. Navigate to the Sign up page
    2. Enter a valid email in the Email field
    3. Enter a valid password in the Password field

Expected result - The user is navigated to the Home Page; No validation error messages displayed

Postconditions - The system has created a new account; The user is logged in
