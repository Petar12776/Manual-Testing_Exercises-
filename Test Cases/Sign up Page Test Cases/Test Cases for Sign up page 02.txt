Test Case ID - TC-16
Scenario ID - SC_04

Title - Verify a Sign up with an invalid email and password

Description - After trying to sign up the website must display an error message telling the user to enter a valid email and password.

Preconditions - User not logged in; User does not have an account

Test Data: email - testingEmailgmailcom; password - my-password12

Test Steps: 
    1. Navigate to the Sign up page
    2. Enter an invalid email in the Email field
    3. Enter an invalid password in the Password field
    4. Click on the Sign up button

Expected result - The website must display an error message to the user

Postconditions - The system doesn't create another account; The user is not logged in
